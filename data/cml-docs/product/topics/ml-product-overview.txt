Cloudera Machine Learning 개요Cloudera Docs Cloudera Machine Learning 개요 기계 학습은 오늘날 현대 기업이 성장하고 경쟁력을 유지하는 데 가장 중요한 기능 중 하나가 되었습니다. 내부 프로세스 자동화부터 소비되는 거의 모든 제품의 디자인, 제작, 마케팅 프로세스 최적화에 이르기까지 ML 모델은 업무와 개인 생활의 거의 모든 측면에 스며들었습니다. ML 개발은 반복적이고 복잡하며, 대부분의 ML 도구가 전체 기계 학습 수명주기에 맞게 구축되지 않았기 때문에 더욱 어려워졌습니다. Cloudera Data Platform의 Cloudera Machine Learning은 데이터 과학자가 모든 AI 사용 사례를 지원하기 위해 모든 것을 포함하는 단일 통합 플랫폼에서 협업할 수 있도록 지원하여 가치 실현 시간을 가속화합니다. 민첩한 실험 및 프로덕션 ML 워크플로를 위해 특별히 제작된 Cloudera Machine Learning은 데이터 준비부터 MLOps, 예측 보고까지 모든 것을 관리합니다. 더 빠른 속도와 민첩성으로 전체 수명주기에 걸쳐 미션 크리티컬 ML 문제를 해결하여 비즈니스에 변화를 가져올 수 있는 기회를 발견하세요. 각 ML 작업 공간을 통해 데이터 과학자 팀은 엔터프라이즈 데이터 클라우드 내에서 관리되는 데이터에 대한 예측 애플리케이션을 구축하기 위한 기계 학습 모델을 개발, 테스트, 교육하고 궁극적으로 배포할 수 있습니다.
